# Student Data Manager

## Project Overview

A simple CRUD (Create, Read, Update, Delete) terminal application built using Python.

This program allows users to:
- Add student data
- Display all students
- Update student information
- Delete student data
- Automatically save student data to a JSON file
- Automatically load saved student data at startup

---

## Technologies Used

- Python
- JSON
- List
- Dictionary

---

## Features

### Create Student
Add a new student with:
- ID
- Name
- Class

---

### Read Student Data
Display all student data stored in the JSON file

---

### Update Student Data
Update:
- Student name
- Student class

based on student ID.

---

### Delete Student Data
Delete student data using student ID.
